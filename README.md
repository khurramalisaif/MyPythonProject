## Project Description
This is a **Python web-based calculator** built using **Streamlit**.  
The application allows users to perform **basic arithmetic operations** such as **addition, subtraction, multiplication, and division** through a **simple and interactive web interface**.

The project demonstrates the use of **functions**, **modular programming**, **input validation**, and **interactive UI development** in Python.

## Features
- Accept **two numerical inputs** from the user  
- Perform arithmetic operations:  
  - **Addition**  
  - **Subtraction**  
  - **Multiplication**  
  - **Division**  
- Display results **instantly**  
- Handle errors such as **division by zero** gracefully  
- Built with **Streamlit** for a user-friendly interface
